# Cube - 异次元杀阵 游戏项目

## 项目概述

本项目是一个完整的游戏工程，旨在实现类似电影《Cube》（异次元杀阵）的游戏功能。游戏类型为**迷宫类MOBA游戏**，玩家将在充满陷阱和谜题的立方体迷宫中展开竞技对战。

## 游戏特色

- **迷宫探索**：玩家需要在不断变化的立方体迷宫中探索和移动
- **MOBA竞技**：结合多人在线战术竞技元素，玩家之间可以进行对战
- **策略解谜**：迷宫中的陷阱和谜题需要玩家运用策略和智慧来解决
- **动态环境**：迷宫结构可能会动态变化，增加游戏的不确定性和挑战性

## 技术架构

### 项目结构

项目采用前后端分离的架构，主要分为三大模块：

```
MyProject/
├── client/          # 客户端（Unity）
├── server/          # 服务端（.NET）
└── doc/            # 项目文档
```

### 客户端（Client）

- **开发引擎**：Unity
- **开发语言**：C#
- **主要职责**：
  - 游戏场景渲染和表现
  - 玩家输入处理
  - UI界面展示
  - 与服务器通信
  - 本地游戏逻辑处理

### 服务端（Server）

- **开发框架**：.NET
- **开发语言**：C#
- **主要职责**：
  - 游戏服务器逻辑
  - 玩家数据管理
  - 房间匹配系统
  - 游戏状态同步
  - 反作弊和安全验证

### 文档（Doc）

- 项目设计文档
- API接口文档
- 开发规范文档
- 部署运维文档
