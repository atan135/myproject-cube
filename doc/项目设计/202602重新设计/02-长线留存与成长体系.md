# 长线留存与成长体系

> 本文档聚焦 **玩家为什么持续玩下去**——从局内成长、局外进度、社交生态到赛季内容更新，构建完整的长效留存框架。

---

## 一、设计原则

### 核心约束

**属性（攻击、生命等）不能带入局内。** 这是保证竞技公平的底线。

因此成长体系必须从"数值堆砌"转向：

```
传统 MOBA 成长：花钱/花时间 → 数值更高 → 打得过新人 ❌
本游戏的成长：花时间 → 解锁更多"可能性" → 面对变化时有更多选择 ✅
```

### 三层留存驱动

| 层级 | 驱动力 | 持续时间 |
|------|--------|---------|
| **短期** | 单局内模块化改造的新鲜组合 | 每局 20-40 分钟 |
| **中期** | 局外图纸解锁、天赋树培养、熟练度变体 | 数周至数月 |
| **长期** | 赛季内容更新、社区事件、叙事揭秘 | 数月至数年 |

## 二、局内成长：Roguelike 构筑体验

### 2.1 模块化改造的"化学反应"

借鉴《Noita》《死亡细胞》《以撒的结合》的核心乐趣——**组合涌现**。

不只给玩家单一词条，而是让 **物理法则 × 身体模块** 产生意想不到的"化学反应"：

| 物理词条 | 身体模块 | 超载组合（Overload）效果 |
|---------|---------|----------------------|
| 极低气压 | 高压喷射腿 | 位移翻倍 + 音爆伤害 |
| 失重 | 磁力抓钩 | 房间内自由飞行 |
| 时间加速 | 过载核心 | 短时间极限爆发（有自毁风险） |
| 禁魔区 | 机械义肢全套 | 物理流派大幅增强 |
| 光影杀机 | 光学迷彩核心 | 在光影切换时触发隐身+伤害脉冲 |

**设计目标**：让玩家在数百小时后仍能发现新的隐藏组合，保持探索的新鲜感。

### 2.2 局内资源博弈

- 击杀中立生物或敌方玩家掉落模块，但 **安装需要拆卸已有部件**
- 稀有模块只在高风险区域（靠近坍缩边缘、Boss 房间）出现
- 玩家之间争夺高价值模块，形成自然的冲突热点
- 模块碎片可以在特定"工坊房间"合成，但合成期间玩家脆弱

## 三、局外成长体系

### 3.1 模块图纸库——解锁"变异可能性"

玩家不能带属性进游戏，但可以带 **"可能性"**。

| 阶段 | 玩家体验 |
|------|---------|
| 新玩家 | 局内击杀只能捡到"基础动力腿"等初级模块 |
| 解锁图纸后 | 有几率在局内刷出"高跳收割腿""引力反转模块"等高级模块 |
| 资深玩家 | 图纸库丰富，面对任何物理词条都有备选方案 |

> **关键平衡**：老玩家的优势是"选择更多"而非"数值更高"。新玩家拿到基础模块依然能打，只是老玩家有更多组合策略。

#### 图纸获取途径

- 比赛掉落（随机概率）
- 成就达成（如"连续 3 局使用纯近战流通关"）
- 赛季通行证奖励
- 社区事件限定图纸

### 3.2 "矩阵契约"——局外天赋树

借鉴 Dead Cells 的升级思路，天赋树 **不加攻击力**，而是影响"操作手感"和"资源效率"。

| 天赋示例 | 效果 | 类型 |
|---------|------|------|
| 废品回收 | 拆卸敌方模块时返还更多能量 | 资源效率 |
| 空间感知 | 房间置换时硬直时间缩短 0.5 秒 | 操作手感 |
| 频率敏感 | 感官借用的持续时间延长 20% | 信息优势 |
| 应急封装 | 断线重连时的保护罩持续时间 +3 秒 | 容错机制 |
| 模块亲和 | 特定类型模块的安装动画加速 | 操作效率 |

> **平衡点**：这些天赋只微调操作手感和资源获取速度，不改变"一刀多少血"的硬性数值，保证了大规模混战时的底层公平。

### 3.3 英雄/核心的"熟练度变体"

孤狼模式下，每个玩家的核心实体可以通过使用习惯 **解锁进化分支**。

```
基础核心
  ├── 近战专精分支 ← 多次使用近战模块后解锁
  │   └── 电击模块产生"弹射"效果
  ├── 远程专精分支 ← 多次使用远程模块后解锁
  │   └── 投掷物自动追踪微调
  └── 生存专精分支 ← 多次在极端环境存活后解锁
      └── 物理词条负面效果减半
```

**策略意义**：玩家有动力深耕某一流派，形成"招牌打法"，同时在直播/社区中展现个人风格。

### 3.4 局外基地建设——"黑客掩体"

玩家在局外经营个人实验室/掩体，提供 **功能性** 而非纯装饰：

| 设施 | 功能 | 解锁条件 |
|------|------|---------|
| 物理模拟室 | 提前体验特定物理词条环境，实战前做准备 | 通关指定次数 |
| 模块研究台 | 查看已收集模块的隐藏属性和组合提示 | 收集一定数量模块 |
| 黑市终端 | 提高开局初始随机池中特定模块类型的出现概率 | 投入代币升级 |
| 情报网络 | 赛季迷宫的部分词条预告 | 社区贡献积分 |
| 训练场 | 与自己的历史"影子数据"对练 | 上传过影子数据 |

## 四、社交系统与社区生态

### 4.1 非对称异步社交

《异次元矩阵》的核心魅力是孤独感，但长线运营需要社交。借鉴《艾尔登法环》的"淡社交"理念：

#### 死者留言与幻影

- 玩家可以在房间墙壁留下编号暗示（或 **故意误导**）
- 死亡玩家的最后几秒以"幻影"形式随机出现在其他玩家的世界中
- 这种间接社交保持了孤独感的同时增加了世界的"有人味"

#### "矩阵屠夫"机制

- 通关一次后，可以将最强的模块化身体作为数据上传
- 这个"影子"会随机出现在其他玩家的迷宫中作为 **精英 Boss**
- 击败它可以获得该玩家生前的稀有模块
- 被击杀的"影子主人"会收到通知和少量奖励

#### 社区共同解密

- 设计需要全服玩家共同提交"矩阵碎片"才能解锁的新维度区域
- 将单机体验升华为社区事件
- 解锁进度全服可见，营造"全服共同攻关"的氛围

### 4.2 组队模式（可选）

- 50-100 人可分为 25-50 支双人小队
- **阵亡队友模组重构**：只要活着的队友拿到阵亡者的"核心模块"，就能在特定房间复活他
- 复活后的队友只保留核心能力，需要重新收集模块

## 五、碎片化叙事——"世界真相"的剥离

玩家持续游玩的深层动力："**这该死的矩阵到底是谁造的？**"

### 5.1 模块记忆残留

每一个捡到的身体模块都可能包含 **前任主人的音频或视觉记忆碎片**。

- 集齐特定套装（如"开拓者序列"）→ 拼凑完整的反转剧情
- 不同套装揭示不同视角的故事
- 有些记忆碎片之间互相矛盾，增加悬念

### 5.2 多重结局

迷宫不应只有一个出口：

| 结局路径 | 条件 | 叙事意义 |
|---------|------|---------|
| **逃离** | 到达出口 | 自由，但矩阵依然运行 |
| **取代** | 极致改造后对抗矩阵 AI | 成为新的规则制定者 |
| **融合** | 放弃人类身份完全机械化 | 与矩阵共生 |
| **毁灭** | 触发自毁序列 | 与矩阵同归于尽 |

不同结局解锁不同的"后日谈"叙事碎片，激励玩家多次通关。

## 六、赛季制——"逻辑规则重构"

### 6.1 规则变异赛季

每个赛季引入新的核心规则变量，迫使玩家重新学习和适应：

| 赛季主题示例 | 规则变化 | 玩家影响 |
|-------------|---------|---------|
| 磁场失控 | 金属模块在磁力房间受损 | 生化模块流派崛起 |
| 虚空裂隙 | 新增传送门房间 | 地图流动性大增 |
| 时间风暴 | 全局随机时间加速/减速 | 节奏把控成为关键技能 |
| 寄生入侵 | 中立生物主动追踪玩家 | PvE 压力增大 |
| 光子衰变 | 全图逐渐变暗至全黑 | 感官借用和声纳成为核心能力 |

### 6.2 每日挑战矩阵

- 每天生成一个 **全服统一的固定迷宫**
- 玩家比拼：最短时间 / 最少置换次数 / 最少改造次数 到达终点
- 排行榜竞争 + 每日奖励
- 为核心玩家提供稳定的日常内容

### 6.3 赛季专属内容

- 新赛季推出新模块图纸、新物理词条、新房间类型
- 限时事件（社区解密、Boss Rush 等）
- 赛季通行证包含限定外观和图纸

## 七、留存漏斗总结

```
首次体验（0-2小时）
  → 核心玩法是否有趣？（房间置换、模块改造新鲜感）
  │
短期留存（1-7天）
  → 每局体验是否不同？（Roguelike 组合、物理词条变化）
  │
中期留存（1-3个月）
  → 图纸解锁、天赋培养、熟练度变体是否有奔头？
  │
长期留存（3个月+）
  → 赛季更新是否带来新鲜感？社区/叙事是否有吸引力？
  │
核心粉丝（1年+）
  → "矩阵屠夫"影子、社区解密、排行榜荣誉
```

> **设计总结**：从"每局的随机性"到"跨局的成长感"到"跨赛季的社区归属"，形成完整的三级留存阶梯。
