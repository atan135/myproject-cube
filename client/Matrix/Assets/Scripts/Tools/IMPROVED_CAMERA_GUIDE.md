# 改进版相机控制器使用说明

## 🆕 新增功能

### 多种缩放模式
解决了原始版本中移动和缩放冲突的问题，提供了三种缩放行为：

1. **Orbit模式（轨道模式）** - 默认
   - 相机围绕一个中心点旋转和缩放
   - 移动时会更新轨道中心点
   - 适合场景浏览和观察

2. **Dolly模式（推拉模式）**
   - 相机直接向前/向后移动
   - 不改变观察目标点
   - 适合精确的位置调整

3. **Combined模式（组合模式）**
   - 智能切换：移动时使用推拉，静止时使用轨道
   - 结合了两种模式的优点
   - 最灵活的使用方式

## 🔧 使用方法

### 基本操作保持不变：
- **WASD**：前后左右移动
- **Shift**：加速移动
- **鼠标右键+拖拽**：旋转视角
- **鼠标滚轮**：缩放

### 新增功能：
- **轨道中心点**：移动时自动更新，缩放围绕此点进行
- **多种缩放模式**：在Inspector中选择合适的模式
- **改进的边界处理**：更合理的边界限制

## ⚙️ Inspector参数说明

### 缩放设置新增：
- **Zoom Behavior**：选择缩放模式（Orbit/Dolly/Combined）
- **Orbit Center**：当前的轨道中心点（运行时显示）

### 边界设置改进：
- 提供了更合理的默认边界值
- 可以更好地限制相机移动范围

## 🎯 推荐配置

### 场景浏览：
- Zoom Behavior: Orbit
- Enable Bounds: true
- 设置合适的边界范围

### 精确编辑：
- Zoom Behavior: Dolly  
- Enable Bounds: false
- 更自由的移动和缩放

### 通用用途：
- Zoom Behavior: Combined
- Enable Bounds: true
- 平衡灵活性和控制性

## 🐛 已解决的问题

1. **移动后无法缩放**：通过分离轨道中心和相机位置解决
2. **缩放干扰移动**：智能模式切换避免冲突
3. **位置跳动问题**：平滑的插值和合理的更新逻辑
4. **边界限制不合理**：提供更好的默认值和可视化

## 🔍 调试信息

运行时会在Scene视图中显示：
- 黄色球体：当前轨道中心点
- 红色线框：边界范围
- 彩色线条：相机朝向指示

控制台会输出详细的处理信息，帮助理解相机行为。

## 💡 使用建议

1. **开始使用时**：建议先用Combined模式，适应后再选择专用模式
2. **场景较大时**：使用Orbit模式配合边界限制
3. **精细调整时**：切换到Dolly模式获得更精确的控制
4. **出现问题时**：查看控制台调试信息定位问题

这个改进版本彻底解决了移动和缩放的冲突问题，提供了更加流畅和直观的相机控制体验。
