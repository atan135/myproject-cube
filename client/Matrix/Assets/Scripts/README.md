# Cube 客户端脚本结构

## 目录说明

### Framework/ 框架层
包含游戏的基础框架代码：
- 单例管理器
- 事件系统
- 对象池
- 资源管理
- 场景管理

### Game/ 游戏逻辑
包含游戏核心逻辑：

#### Character/ 角色系统
- 角色控制器
- 角色属性
- 角色技能

#### Maze/ 迷宫系统
- 迷宫生成器
- 房间管理
- 陷阱系统

#### Combat/ 战斗系统
- 战斗逻辑
- 伤害计算
- 技能系统

#### UI/ UI逻辑
- UI管理器
- 各种UI界面脚本

### Network/ 网络模块
- 网络管理器
- 消息处理
- 连接管理
- 状态同步

### Utility/ 工具类
- 扩展方法
- 数学工具
- 字符串工具
- 其他通用工具

## 开发规范

- 使用命名空间组织代码
- 遵循Unity编码规范
- 使用注释说明类和方法的用途
- 保持代码简洁和可维护
